# vyevd-dotfiles

Rice: Bocchi the Rock / Ryo Yamada  
Navy `#0d1520` — Gold `#e6d47b` — Kita red `#bb607b`

Arch Linux + Hyprland + Kitty + Neovim + Waybar + Rofi + Starship

## What's tracked

- `hypr/` — window manager config
- `kitty/` — terminal emulator
- `waybar/` — status bar
- `bashrc` — shell
- `starship.toml` — prompt
- `fastfetch/` — system info
- `rofi/` — app launcher + powermenu
- `nvim/` — editor
- `mpv/` — video player
- `local/bin/verse` — random bible verse on terminal open
